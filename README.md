# Developers guide

1. Get someone to add your gmail account to mobile-era-2018 at Firebase
2. Install NodeJS
3. Install Firebase tools: `npm install -g firebase-tools`
4. Log in to Firebase: `firebase login`
5. Deploy the new version `firebase deploy`
